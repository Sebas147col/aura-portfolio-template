## 🇺🇸 English Version

Welcome to **Aura**! This is an ultra-premium portfolio template specifically designed for developers, designers, and modern creators looking to capture the attention of clients and recruiters instantly.

This template is fully optimized, ready to be customized, and published on **Gumroad** or other creator platforms.

### 🎨 Key Features
*   **Premium Visual Aesthetics:** Inspired by Apple and Stripe visual designs, featuring a deep dark slate color scheme, vibrant color accents, and modern typography.
*   **Glassmorphism Effect:** Interactive cards and components styled with a translucent frosted-glass effect using modern CSS properties.
*   **Custom Interactive Cursor:** An animated outer ring that smoothly follows the mouse cursor and expands/changes style when hovering over interactive elements.
*   **Ambient Background Glows:** Subtle floating gradient orbs in the background animated via pure CSS keyframes.
*   **Scroll-Reveal Animations:** Page elements animate smoothly into view as you scroll down, powered by the JavaScript `IntersectionObserver` API.
*   **Interactive Experience Timeline:** A clean work experience timeline with active line indicators.
*   **Functional Contact Form (Mock):** Form validation setup with an animated CSS toast notification on successful submission.
*   **Fully Responsive:** 100% responsive design optimized for mobile devices, tablets, and high-resolution monitors.

### 🛠️ Tech Stack
*   **Semantic HTML5:** Clean and SEO-friendly layout structure.
*   **Vanilla CSS3:** 100% editable and lightweight styles. No heavy CSS framework dependencies (like Bootstrap or Tailwind).
*   **Modern JavaScript:** Seamless custom animations, cursor tracking, and scroll-triggered events.
*   **Lucide Icons:** Modern vector icon library loaded dynamically.
*   **Vite:** Fast local dev server with hot-reload and bundling configuration.

### 🚀 Getting Started (Local Development)
To run this template locally and view changes in real-time as you edit:
1.  Make sure you have [Node.js](https://nodejs.org/) installed.
2.  Install local development dependencies:
    ```bash
    npm install
    ```
3.  Start the local dev server:
    ```bash
    npm run dev
    ```
4.  Open your browser and navigate to the address shown in the terminal (usually `http://localhost:5173`).

#### 📦 Build for Production
To compile and minify all assets, ready for deployment to any free hosting provider (like Vercel, Netlify, or GitHub Pages):
```bash
npm run build
```
This will generate a `dist` folder with production-ready code.

## 🇪🇸 Versión en Español

¡Bienvenido a **Aura**! Esta es una plantilla de portafolio ultra-premium diseñada específicamente para desarrolladores, diseñadores y creadores modernos que buscan captar la atención de clientes y reclutadores de inmediato.

Esta plantilla está completamente optimizada, lista para ser personalizada y publicada en **Gumroad** u otras plataformas de creadores.

### 🎨 Características Destacadas
*   **Estética Visual Premium:** Inspirada en el estilo visual de Apple y Stripe con una paleta de colores oscuros profundos, acentos de color vibrantes e tipografías modernas.
*   **Efecto Glassmorphism:** Tarjetas y componentes interactivos diseñados con el efecto de vidrio translúcido mediante propiedades CSS modernas.
*   **Cursor Interactivo Personalizado:** Un aro exterior animado que persigue el cursor del ratón y reacciona de forma fluida cuando se pasa sobre elementos clicables.
*   **Luces Ambientales de Fondo:** Globos de colores degradados que flotan de forma sutil en el fondo mediante animaciones de fotogramas clave (keyframes) en CSS puro.
*   **Animaciones Scroll-Reveal:** Los elementos de la página aparecen progresivamente mediante efectos de aparición suaves basados en la API `IntersectionObserver` de JavaScript.
*   **Sección de Experiencia Interactiva:** Una cronología de trabajo ordenada y limpia con indicador de línea activa.
*   **Formulario de Contacto Funcional (Mock):** Formulario integrado con validaciones completas y aviso de confirmación tipo "Toast" animado.
*   **Totalmente Responsivo:** Adaptado al 100% para dispositivos móviles, tabletas y monitores de alta resolución.

### 🛠️ Tecnologías Utilizadas
*   **HTML5 Semántico:** Estructura limpia y accesible de acuerdo a las directrices de SEO.
*   **Vanilla CSS3:** Sin dependencias externas de frameworks pesados (como Bootstrap o Tailwind). Estilos 100% editables y veloces.
*   **Modern JavaScript:** Animaciones personalizadas, control de cursor dinámico y efectos interactivos.
*   **Lucide Icons:** Conjunto de iconos vectoriales modernos y estilizados cargados dinámicamente.
*   **Vite:** Servidor de desarrollo súper rápido para recarga en tiempo real y empaquetamiento listo para producción.

### 🚀 Inicio Rápido (Desarrollo Local)
Para ejecutar esta plantilla de forma local y ver los cambios al instante mientras la editas:
1.  Asegúrate de tener instalado [Node.js](https://nodejs.org/).
2.  Instala las dependencias del servidor de desarrollo local:
    ```bash
    npm install
    ```
3.  Inicia el servidor local:
    ```bash
    npm run dev
    ```
4.  Abre tu navegador en la dirección local indicada por la terminal (usualmente `http://localhost:5173`).
